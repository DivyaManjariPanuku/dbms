AGGREGATE FUNCTION

There are 5 functions

  • count()
  • sum()
  • min()
  • max()
  • avg()

.

Count()

The count function is used to count the data rws returned in the result set.

Sum()

The sum function is used to return the total sum of a given numeric column.The sum function will only work on numeric datatype.

Min()

The min function is used to return the smallest value of a given numeric column.The min function will only work on numeric datatype as well as string columns.

Max()

The max function is used to return the largest value of a given numeric column.The max function will only work on numeric datatype as well as string columns.

avg()

The avg or average function is used to return the average value of a given numeric column.The avg function will only work on numeric datatype.
