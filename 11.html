ABS()

This Numeric Function is used to convert (-VE) value into (+VE) value.

Syntax: ABS(number)


CEIL()

This Numeric Function is used to return a value that is greater than or equal to the given value.

Syntax: CEIL(NUMBER)


FLOOR()

This numeric function is used to return the largest integer equal to or less than n.

Syntax: FLOOR(NUMBER)


MOD()

The MOD() function is used to return the remainder value.

Syntax: MOD(m, n)


POWER()

This numeric function is used to return the power of a given expression.

Syntax: POWER(m, n)


ROUND()

This numeric function returns the nearest value of the given expression.

Syntax: ROUND(NUMBER,[DECIMAL PLACES])


TRUNC()

The TRUNC (number) function is used to return n1 truncated to n2 decimal places.

Syntax: TRUNC (NUMBER, DECIMAL PLACES)
