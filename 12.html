LENGTH()

The String LENGTH function is used to return the length of a given string.


LOWER()

The string LOWER() function is used to return a specified character expression in lowercase letters.

Syntax: LOWER(CharExpression)


UPPER()

The string UPPER() function is used to return a specified character expression in uppercase letters.

Syntax: UPPER(CharExpression)


CONCAT()

The string CONCAT() function is used to return the result (a string) of concatenating two string values.

Syntax: CONCAT(char1, char2)
