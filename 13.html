SYSDATE

The SYSDATE function is used to return the current date and time set for the operating system on which the database resides.


ADD_MONTHS()

The ADD_MONTHS date function is used to return the date with a given number of months added (date plus integer months).

Syntax: ADD_MONTHS(date, integer)


LAST_DAY()

The LAST_DAY() date function is used to return the last day of the month that contains a date.

Syntax: LAST_DAY(date)

MONTHS_BETWEEN()

The MONTHS_BETWEEN() date function is used to get the number of months between two dates (date1, date2).

Syntax: MONTHS_BETWEEN(DATE1, DATE2)
